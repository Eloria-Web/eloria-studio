# Eloria Studio

Plataforma de suscripción premium para contenido creativo.

## Características

- Autenticación con Supabase
- Integración con PayPal y Stripe
- Dominio personalizado: studio.eloria.paris
- Sistema de suscripciones Premium

## Configuración

Este proyecto está configurado para desplegarse automáticamente en Netlify.

## Tecnologías

- HTML5
- CSS3
- JavaScript
- Supabase (Backend)
- PayPal & Stripe (Pagos)
- Netlify (Hosting)
